# Syllabus
vil = villageois

# Shortcuts
From vil: 
- House: Q A
- Woodshop: Q R
- Moulin: Q Z
- Ecurie: S E

Group
- n: 1,2,3, etc.. 
- Ctrl+n: Create group n
- n: select group n
- Shift+n: add selection to group n

Misc
- shift click: allow to chain actions
